# agent.md

本项目的 Agent 协作规范正式维护在同目录的 `AGENTS.md` 中。

请优先阅读：

- `./AGENTS.md`
- `./docs/architecture/project-architecture.md`

之所以使用 `AGENTS.md` 作为主文件，是为了兼容常见的 Agent / Codex 工作流约定，方便后续工具自动发现并执行其中的开发规范。

其中：

- `AGENTS.md` 负责定义完整开发流程。
- `docs/architecture/project-architecture.md` 负责定义当前项目级架构图、模块边界和维护规则。
